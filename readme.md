# Euler solution in typescript

```shell
Usage: tsx src/index.ts [options]
Options:
      --version  Show version number                     [boolean]
  -c, --create   Create new problem file                  [number]
  -r, --run      Run solution                             [number]
  -f, --force    Force create new problem                [boolean]
  -a, --all      Run all solution                        [boolean]
      --help     Show help                               [boolean
```
